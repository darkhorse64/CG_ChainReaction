read color:word(1)

gameloop
loop 8 read cells:string(16)
write random

INPUT
color: your color (r=red, b=blue)
cells: 16-char string, 2 chars per cell — ".." (empty), "r1"-"r3" (red), "b1"-"b3" (blue)

OUTPUT
e.g. 3 5 (place an orb on row 3, column 5)
or: random (let the referee pick a random valid move)
